# AI Engineering Skill Library

A production-quality collection of reusable AI engineering skills.

Rather than isolated prompts, this repository provides structured engineering workflows that can be combined to solve complex software engineering tasks consistently.

Each skill follows a common specification.

---

## Skill Structure

Every skill contains:

- Metadata
- Purpose
- When to Use
- Objectives
- Workflow
- Best Practices
- Common Pitfalls
- Output Format
- Validation Checklist
- Success Criteria

---

## Principles

Every skill should:

- Be deterministic
- Avoid unnecessary assumptions
- Explain decisions
- Validate outputs
- Prefer maintainability over cleverness
- Encourage incremental improvement
